# awesome-data-tools

A curated list of data engineering tools and libraries.

## Installation

```bash
pip install awesome-data-tools
```

## Usage

```python
from awesome_data_tools import DataPipeline
pipeline = DataPipeline("config.yaml")
pipeline.run()
```
